## Table of contents

## Information
CookBook is a console application for managing recipes. User can do CRUD operations on recipes. User can also add the recipe to the list of favorite recipes and save it to a JSON or XML file.

## Technologies
* .NET Core 3.1
* Console app
* xUnit
